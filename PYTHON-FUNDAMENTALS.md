------------------------------------------------------------
WHAT IS PYTHON ?
------------------------------------------------------------
PYTHON IS A:
- HIGH LEVEL PROGRAMMING LANGUAGE
- INTERPRETED LANGUAGE
- DYNAMICALLY TYPED LANGUAGE
- GENERAL PURPOSE LANGUAGE

HIGH LEVEL:
- CLOSE TO ENGLISH
- EASY FOR HUMANS TO READ AND WRITE

INTERPRETED:
- CODE IS EXECUTED LINE BY LINE
- NO SEPARATE COMPILATION STEP

DYNAMICALLY TYPED:
- NO NEED TO DECLARE DATA TYPES
- TYPE IS DECIDED AT RUNTIME

GENERAL PURPOSE:
- CAN BE USED IN MANY DOMAINS

------------------------------------------------------------

3) SELECT "ADD PYTHON TO PATH"
4) COMPLETE INSTALLATION

------------------------------------------------------------
PYTHON SYNTAX & INDENTATION
------------------------------------------------------------
- PYTHON DOES NOT USE {} BRACES
- PYTHON USES INDENTATION
- INDENTATION DEFINES CODE BLOCKS

CORRECT EXAMPLE:
--------------------------------
if True:
    print("PYTHON")
--------------------------------

WRONG EXAMPLE:
--------------------------------
if True:
print("ERROR")
--------------------------------

NOTE:
- INDENTATION ERRORS CAUSE PROGRAM TO FAIL

------------------------------------------------------------
VARIABLES IN PYTHON
------------------------------------------------------------
VARIABLE:
- NAME THAT STORES DATA

NO TYPE DECLARATION REQUIRED

EXAMPLE:
--------------------------------
x = 10
name = "PYTHON"
price = 99.5
--------------------------------

PYTHON DECIDES TYPE AUTOMATICALLY


------------------------------------------------------------
VARIABLE NAMING CONVENTIONS
------------------------------------------------------------
RULES:
- MUST START WITH LETTER OR _
- CANNOT START WITH NUMBER
- NO SPACES ALLOWED
- CASE SENSITIVE

VALID NAMES:
--------------------------------
name
_total
student_age
--------------------------------

INVALID NAMES:
--------------------------------
1name
my name
total-value
--------------------------------

NAMING STYLE:
snake_case (RECOMMENDED)


------------------------------------------------------------
PRINT() FUNCTION
------------------------------------------------------------
PRINT() IS USED TO DISPLAY OUTPUT

EXAMPLES:
--------------------------------
print("HELLO WORLD")
print(10)
print(10 + 20)
--------------------------------

PRINT MULTIPLE VALUES:
--------------------------------
print("AGE IS", 20)
--------------------------------


------------------------------------------------------------
INPUT & OUTPUT
------------------------------------------------------------
INPUT IS TAKEN USING input()

EXAMPLE:
--------------------------------
name = input("ENTER YOUR NAME: ")
print("HELLO", name)
--------------------------------

IMPORTANT POINT:
- input() RETURNS STRING BY DEFAULT

TYPE CONVERSION:
--------------------------------
age = int(input("ENTER AGE: "))
print(age + 1)
--------------------------------
